# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/13b5c890-77f7-4a98-b843-65a7feb3e197

## Run Locally

**Prerequisites:**  Node.js


Quick Clone + Run Steps

1. Open Command Prompt or PowerShell.

2. Clone the repository:
git clone <repository-url>

3. Open the cloned folder in VS Code.

4. In the VS Code terminal, run:
npm install

5. Start the project:
npm run dev

Alternative startup commands if needed:
npm run build
npm run start

6. Open the local URL shown in the terminal (usually http://localhost:3000)

7. If the port is busy, stop the server with Ctrl + C and run the command again.
